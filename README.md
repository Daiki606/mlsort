# mlsort

`mlsort` は、標準入力から受け取った複数行のデータを自動的に解析し、  
**データの種類に応じた最適な方法で整列（ソート）するコマンド**である。

本コマンドは、入力されたデータが以下のどの形式であるかを自動判別する：

- **整数（int）**
- **浮動小数点数（float）**
- **日付（YYYY-MM-DD形式）**
- **JSONオブジェクト（1行1 JSON）**
- **テキスト（上記のどれにも当てはまらない場合）**

判別されたデータ型に合わせて、最適な基準でソートを行う。  
UNIX らしい標準入出力ベースのインターフェースを持ち、  
自動的に振る舞いが変わる “smart” なコマンドである。

---

## 特徴

- 入力の内容を自動解析してソート方式を切り替える  
- int / float / date / json / text を自動認識  
- 標準入力（stdin）と標準出力（stdout）を用いた UNIX 互換の使い方  
- シンプルな実装でありつつ、柔軟な挙動を実現  
- GitHub Actions によるテストをサポート
